## 16.1 Ser – estar

### 16.1.1 Principios básicos

#### 16.1.1.1 Definición general

Una de las particularidades más llamativas del español es que dispone de dos verbos, *ser* y *estar*, para expresar significados correspondientes a los verbos únicos *zijn* (NL), *to be* (EN) o *être* (FR). Los dos verbos *ser* y *estar* tienen una serie de usos diversos aunque relacionados. El denominador común de la mayoría de los usos de *ser* es que *definen* o *identifican* el sujeto al que se aplican. Los usos de *estar*, en cambio, describen dónde y cómo se encuentra el sujeto.

#### 16.1.1.2 Usos prototípicos de *ser*

Utilizamos el verbo *ser* para **definir a alguien o algo**, en el sentido de **describir las características**. En el caso de una persona, puede tratarse de:

| | |
| --- | --- |
| origen | *es belga* |
| talla | *es grande, pequeño* |
| color de pelo o de piel (natural) | *es rubio, es blanco* |
| profesión | *es profesor / médico* |
| género y orientación sexual | *es mujer, es heterosexual* |

En el caso de objetos, o experiencias puede tratarse de:

| | |
| --- | --- |
| características físicas | <ul><li>*la iglesia es muy alta*</li><li>*un balón tiene que ser redondo*</li><li>*la mesa es de madera*<li>*mi coche es negro*</li></ul> |
| características emocionales | <ul><li>*la experiencia fue muy dura*</li><li>*la película es muy divertida*</li></ul> |

Otra manera de definir un objeto o una persona es decir a qué categoría pertenece, o de qué individuo se trata:
- *ella es mi tía*
- *ella es Rosa*
- *Antonia es mi esposa*
- *mi coche es un Mercedes*
- *¿Qué es esto? Es un sacacorchos.*

#### 16.1.1.3 Usos prototípicos de *estar*

El primer uso prototípico de *estar* consiste en describir el estado en que se encuentra una persona o un objeto:
- *Pedro está enfermo / sano / triste / contento.*
- *La casa está limpia / sucia / ordenada.*

Luego, *estar* sirve para localizar objetos o personas:
- *¿Dónde está Juan?*
- *La iglesia está a mano derecha.*
- *Los cuadernos están en el cajón.*
